{ إِنَّمَا يُرِيدُ الله لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) }

{ قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }

{ فَقُلْ تَعَالَوْا نَدْعُ أَبْنَاءَنَا وَأَبْنَاءَكُمْ وَنِسَاءَنَا وَنِسَاءَكُمْ وَأَنفُسَنَا ... ( آل عمران : 61 ) }

{ إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) }

{ وَأَنذِرْ عَشِيرَتَكَ الْأَقْرَبِينَ ( الشعراء : 214 ) }

{ إِنَّمَا وَلِيُّكُمُ الله وَرَسُولُهُ وَالَّذِينَ آمَنُوا الَّذِينَ يُقِيمُونَ الصَّلَاةَ وَيُؤْتُونَ ... ( المائدة : 55 ) }

{ إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ أُولَئِكَ هُمْ خَيْرُ الْبَرِيَّةِ ( البينة : 7 ) }

8 { يَا أَيُّهَا الرَّسُولُ بَلِّغْ مَا أُنزِلَ إِلَيْكَ مِن رَّبِّكَ وَإِن لَّمْ تَفْعَلْ فَمَا بَلَّغْتَ رِسَالَتَهُ ( المائدة : 67 ) }

{ الْيَوْمَ أَكْمَلْتُ لَكُمْ دِينَكُمْ وَأَتْمَمْتُ عَلَيْكُمْ نِعْمَتِي ( المائدة : 3 ) }

{ وَالسَّابِقُونَ السَّابِقُونَ @ أُولَئِكَ الْمُقَرَّبُون ( الواقعة : 10 - 11 ) }

{ وَمِنَ النَّاسِ مَن يَشْرِي نَفْسَهُ ابْتِغَاءَ مَرْضَاتِ الله وَالله رَءُوفٌ بِالْعِبَادِ ( البقرة : 207 ) }

{ أَفَمَن كَانَ مُؤْمِنًا كَمَن كَانَ فَاسِقًا لَّا يَسْتَوُونَ ( السجدة : 18 ) }

{ وَإِنِّي لَغَفَّارٌ لِّمَن تَابَ وَآمَنَ وَعَمِلَ صَالِحًا ثُمَّ اهْتَدَى( طه : 82 ) }

{ فَاسْأَلُوا أَهْلَ الذِّكْرِ إِن كُنتُمْ لَا تَعْلَمُونَ ( النحل : 43 ) }

{ يَا أَيُّهَا الَّذِينَ آمَنُوا اتَّقُوا الله وَكُونُوا مَعَ الصَّادِقِينَ ( التوبة : 119 ) }

{ وَقِفُوهُمْ ۖ إِنَّهُم مَّسْئُولُونَ ( الصافات : 24 ) }

{ هَذَانِ خَصْمَانِ اخْتَصَمُوا فِي رَبِّهِمْ ( الحج : 19 ) }

{ وَتَعِيَهَا أُذُنٌ وَاعِيَةٌ ( الحاقة : 12 ) }

{ وَمَنْ عِندَهُ عِلْمُ الْكِتَابِ ( الرعد : 43 ) }

{ أَجَعَلْتُمْ سِقَايَةَ الْحَاجِّ وَعِمَارَةَ الْمَسْجِدِ الْحَرَامِ كَمَنْ آمَنَ بِالله ... ( التوبة : 19 ) }

{ وَإِن تَظَاهَرَا عَلَيْهِ فَإِنَّ الله هُوَ مَوْلَاهُ وَجِبْرِيلُ وَصَالِحُ الْمُؤْمِنِينَ ( التحريم : 4 ) }

{ سَلَامٌ عَلَى إِلْ يَاسِينَ ( الصافات : 130 ) }

{ وَعَلَى الْأَعْرَافِ رِجَالٌ يَعْرِفُونَ كُلًّا بِسِيمَاهُمْ ( الأعراف : 46 ) }

{ وَمَن يَقْتَرِفْ حَسَنَةً نَّزِدْ لَهُ فِيهَا حُسْنًا ( الشورى : 23 ) }

{ إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ سَيَجْعَلُ لَهُمُ الرَّحْمَنُ وُدًّا ( مريم : 96 ) }

{ سَأَلَ سَائِلٌ بِعَذَابٍ وَاقِعٍ ( المعارج : 1 ) }

{ يَا أَيُّهَا الَّذِينَ آمَنُوا }

{ أَفَمَن كَانَ عَلَى بَيِّنَةٍ مِّن رَّبِّهِ وَيَتْلُوهُ شَاهِدٌ مِّنْهُ ( هود : 17 ) }

{ وَيُطْعِمُونَ الطَّعَامَ عَلَى حُبِّهِ مِسْكِينًا وَيَتِيمًا وَأَسِيرًا ( الانسان : 8 ) }

{ وَاعْتَصِمُوا بِحَبْلِ الله جَمِيعًا وَلَا تَفَرَّقُوا ( آل عمران : 103 ) }

{ وَلَتَعْرِفَنَّهُمْ فِي لَحْنِ الْقَوْلِ ( محمد : 30 ) }

{ يَا أَيُّهَا الَّذِينَ آمَنُوا إِذَا نَاجَيْتُمُ الرَّسُولَ فَقَدِّمُوا بَيْنَ يَدَيْ نَجْوَاكُمْ صَدَقَةً ( المجادلة : 12 ) }

{ الَّذِينَ يُنفِقُونَ أَمْوَالَهُم بِاللَّيْلِ وَالنَّهَارِ سِرًّا وَعَلَانِيَةً ( البقرة : 274 ) }

{ وَأَذَانٌ مِّنَ الله وَرَسُولِهِ إِلَى النَّاسِ يَوْمَ الْحَجِّ الْأَكْبَرِ ... ( التوبه : 3 ) }

{ فَمِنْهُم مَّن قَضَى نَحْبَهُ وَمِنْهُم مَّن يَنتَظِرُ ( الأحزاب : 23 ) }

{ وَكَفَى الله الْمُؤْمِنِينَ الْقِتَالَ ( الأحزاب : 25 ) }

{ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ طُوبَى لَهُمْ وَحُسْنُ مَآبٍ ( الرعد : 29 ) }

{ وَالَّذِي جَاءَ بِالصِّدْقِ وَصَدَّقَ بِهِ ( الزمر : 33 ) }
